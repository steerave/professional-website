**Goal:** Fully reskinn the portfolio site with a Dark Craft / Amber aesthetic — Anybody + DM Mono typography, amber accent replacing blue, ghost-type hero — while updating AI Projects from 2 to 3 cards with new content.

**Architecture:** All changes are component-level reskins on the existing Evolved Linear layout. "use client";

import { useEffect, useRef, useState } from "react";

const stats = [
  {
    numeric: 20,
    suffix: "+",
    line1: "Years Leading",
    line2: "Digital Execution",
  },
  {
    numeric: 30,
    suffix: "+",
    line1: "National-Scale",
    line2: "Programs Delivered",
  },
  {
    numeric: 10,
    suffix: "M+",
    line1: "Users Supported",
    line2: "in Live Systems",
  },
];

function useCountUp(target: number, active: boolean) {
  const [current, setCurrent] = useState(0);
  const hasRun = useRef(false);

  useEffect(() => {
    if (!active || hasRun.current) return;
    hasRun.current = true;

    const duration = 1400;
    const start = performance.now();

    function tick(now: number) {
      const elapsed = now - start;
      const progress = Math.min(elapsed / duration, 1);
      const eased = 1 - Math.pow(1 - progress, 3);
      setCurrent(Math.round(eased * target));
      if (progress < 1) requestAnimationFrame(tick);
    }

    requestAnimationFrame(tick);
  }, [active, target]);

  return current;
}

export default function CredibilityStrip() {
  const ref = useRef<HTMLDivElement>(null);
  const [active, setActive] = useState(false);

  useEffect(() => {
    const el = ref.current;
    if (!el) return;
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setActive(true);
          observer.unobserve(el);
        }
      },
      { threshold: 0.5 }
    );
    observer.observe(el);
    return () => observer.disconnect();
  }, []);

  return (
    <section
      aria-label="Key achievements"
      className="border-y border-border bg-surface"
    >
      <div
        ref={ref}
        className="mx-auto flex max-w-content flex-col items-center gap-10 px-6 py-14 md:flex-row md:gap-0 md:px-10 lg:px-20"
      >
        {stats.map((stat, i) => (
          <div
            key={stat.line1}
            className={`flex-1 text-center ${
              i < stats.length - 1
                ? "md:border-r md:border-border"
                : ""
            }`}
          >
            <StatNumber stat={stat} active={active} />
          </div>
        ))}
      </div>
    </section>
  );
}

function StatNumber({
  stat,
  active,
}: {
  stat: (typeof stats)[number];
  active: boolean;
}) {
  const count = useCountUp(stat.numeric, active);

  return (
    <div className="flex flex-col items-center gap-2">
      <div className="flex items-baseline">
        <span
          className="font-mono font-medium leading-none text-accent"
          style={{ fontSize: "clamp(36px, 5vw, 48px)" }}
        >
          {count}
        </span>
        <span
          className="ml-0.5 font-mono font-medium leading-none text-accent"
          style={{ fontSize: "clamp(20px, 2.5vw, 26px)" }}
        >
          {stat.suffix}
        </span>
      </div>
      <div className="mt-1 flex flex-col items-center">
        <span
          className="font-sans uppercase text-text-muted"
          style={{ fontSize: "11px", letterSpacing: "2px", fontWeight: 300 }}
        >
          {stat.line1}
        </span>
        <span
          className="font-sans uppercase text-text-muted"
          style={{ fontSize: "11px", letterSpacing: "2px", fontWeight: 300 }}
        >
          {stat.line2}
        </span>
      </div>
    </div>
  );
}

# Design System v2 — Dark Craft / Amber

## Aesthetic Direction

Dark, warm, builder's workshop. Amber/gold accent replaces blue. Wide-set Anybody typography communicates bold, technical confidence without corporate polish. DM Mono adds craft and precision to labels and numbers.

Inspired by: workshop tools, amber glow of incandescent light, precision instruments.

## Color Tokens

| Token (CSS var) | Value | Tailwind class | Usage |
|---|---|---|---|
| `--color-bg` | `#080808` | `bg-bg` | Page background |
| `--color-surface` | `#111111` | `bg-surface` | Cards, panels |
| `--color-elevated` | `#1a1a1a` | `bg-elevated` | Hover states |
| `--color-border` | `#1e1e1e` | `border-border` | Dividers |
| `--color-text-primary` | `#ffffff` | `text-text-primary` | Headings |
| `--color-text-body` | `#999999` | `text-text-body` | Body copy |
| `--color-text-muted` | `#444444` | `text-text-muted` | Labels, secondary |
| `--color-accent` | `#ff9f2f` | `text-accent`, `bg-accent`, `border-accent` | Primary amber |
| `--color-accent-light` | `#ffb347` | `text-accent-light` | Lighter amber, hover |

**Never use:** `rgba(74,111,165,...)` — that is the v1 blue. All glow/tint values use `rgba(255,159,47,...)`.

## Typography

| Role | Font | Weight | Class / Style |
|---|---|---|---|
| Display headings | Anybody | 900 | `style={{ fontFamily: "var(--font-anybody)", fontWeight: 900 }}` |
| Subheadings | Anybody | 700 | `fontWeight: 700` |
| Body | Anybody | 300 | `fontWeight: 300` |
| Emphasis | Anybody | 400 | `fontWeight: 400` |
| Technical labels | DM Mono | 400–500 | `className="font-mono"` |

**Section labels:** DM Mono, `fontSize: "10px"`, `letterSpacing: "3px"`, `className="text-accent"`, uppercase.
**Section headings:** Anybody 900, `fontSize: "clamp(32px, 5vw, 48px)"`, `letterSpacing: "-1px"`, uppercase.
**Body:** Anybody 300, 14–17px, `lineHeight: 1.7–1.8`, `text-text-body`.

## Component Patterns

### CTA Buttons
Primary: `bg-accent text-black font-bold` + `padding: "12px 24px"`, `borderRadius: "2px"`, `fontSize: "11px"`, uppercase.
Secondary: `border border-elevated text-text-muted font-light` same sizing.

### Ambient Glow
```tsx
style={{
  background: "radial-gradient(circle, rgba(255,159,47,0.10) 0%, transparent 65%)"

```
Max one per section. Used in Hero (top-right) and CTA (bottom-center).

## Do Not
- No blue values (`rgba(74,111,165,...)`, `#4A6FA5`, `#5a8abf`)
- No serif fonts (Newsreader removed)
- No rounded-lg — use `rounded-sm` (border-radius: 2px) for the sharp, craft feel
