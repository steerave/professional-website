### AI-Powered Task Manager

What it does  
A CLI task manager that bridges the terminal and Obsidian, using natural language parsing to capture, organize, and review tasks without friction.

How it works  
- Natural language input — type a task in plain English and it extracts dates, priorities, and categories automatically
- Two-way Obsidian sync — tasks are stored as markdown files in your vault; changes in Obsidian sync back
- Smart daily summaries — consolidated views across today, this week, and next week
- Extensible taxonomy — define canonical tags and domains for consistent organization

Why it matters  
Shows how thoughtful system design can eliminate friction between task capture and task review — no proprietary lock-in, just plain markdown and a fast CLI.
